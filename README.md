# wilton_test_server
Тестовый сервер на основе библиотеки wilton_server. С использованием wilton_server C API.

### Положение директории
Предполагается следующее расположение:
```
  some_catalog
    wilton
      ...
    wilton_test_server
      ...
  
```
### Сборка с ключами -g -Og

```bash
mkdir build
cd build
cmake .. -DCMAKE_BUILD_TYPE=Debug
```
